Task to solve
=============

Evaluate (sqrt 4), (sqrt 2), and (sqrt -1) in DrScheme. Then, find out whether
DrScheme knows an operation for determining the tangent of an angle.


Solution
========

Evaluate
--------

:
    >(sqrt 4)
    2
    >(sqrt 2)
    1.4142135623730951
    >(sqrt -1)
    0+1i

The tangent of an angle
-----------------------

:
    >(tan 45)
    1.6197751905438615
